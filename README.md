# 30 Days of COBOL — Practical Systems Challenge

![Logo](https://github.com/Cati94/cobol-daily-labs/blob/main/cobol30dias.png)


Este repositório é dedicado a **30 dias consecutivos de desenvolvimento prático em COBOL**, com foco em construir sistemas cada vez mais próximos de cenários empresariais reais.

O objetivo principal é desenvolver competências em:

- Estrutura formal COBOL  
- Manipulação de ficheiros (sequenciais e indexados)  
- Processamento batch  
- Modelação de dados  
- Engenharia procedural  
- Organização modular de código  

---

## Estrutura do Repositório

```
cobol-30-days/
│
├── day01-invoice-generator/
├── day02-payroll-calculator/
├── day03-atm-simulator/
...
├── day30-mainframe-job-simulation/
└── README.md
```

Cada pasta contém:

- Código fonte (`.cob`)  
- README técnico do exercício  
- Explicação da arquitetura  
- Limitações identificadas  
- Sugestões de melhorias futuras  

---

## Progressão Técnica

| Semana | Conteúdos |
|--------|------------|
| 1      | Fundamentos do COBOL e cálculos básicos |
| 2      | Manipulação de ficheiros sequenciais |
| 3      | Processamento batch e relatórios |
| 4      | Engenharia empresarial e introdução a ficheiros INDEXED |

---

## Objetivos ao Final do Desafio

- Domínio de ficheiros sequenciais e atualização por reescrita  
- Criação de relatórios formatados  
- Geração e análise de logs  
- Processamento batch completo  
- Introdução prática a ficheiros **INDEXED**  

---

## Requisitos

- GNUCobol 3.x ou superior  

---

## Como Compilar

```bash
cobc -x program.cob -o program
```

---

## Como Executar

```bash
./program
```
