# 📅 Dia 18 — Sistema Bancário Interativo

## 🎯 Objetivo
Criar um sistema bancário simples em COBOL com menu interativo, permitindo manipular contas e guardar alterações em ficheiro.

Este exercício introduz conceitos de **programação interativa** e **persistência de dados**.

---

## 📂 Ficheiro de dados

`accounts.dat`

Formato:
ACCOUNT-ID BALANCE

Exemplo:
1001 1000
1002 500
1003 200

---

## ⚙️ Funcionalidades

O sistema apresenta um menu:

1 - Listar contas  
2 - Depositar  
3 - Levantar  
4 - Sair
