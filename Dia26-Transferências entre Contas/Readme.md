# 🔁 Dia 26 - Transferências entre Contas (COBOL)

## 📌 Descrição
Implementação de transferências bancárias entre contas.

---

## 🚀 Funcionalidades

- Transferência entre contas
- Validação de saldo
- Atualização de duas contas
- Registo duplo (débito + crédito)

---

## 🧠 Conceitos

- Consistência de dados
- Ficheiros temporários
- Operações multi-registo
- Simulação de transações reais
