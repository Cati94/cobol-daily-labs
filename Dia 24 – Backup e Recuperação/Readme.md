# 💾 Dia 24 - Backup e Recuperação (COBOL)

## 📌 Descrição
Este módulo implementa backup e recuperação de dados no sistema bancário.

---

## 🚀 Funcionalidades

- Backup de contas
- Backup de transações
- Restauração de dados
- Proteção contra perda de informação

---

## 📁 Ficheiros

| Original         | Backup             |
|----------------|------------------|
| accounts.dat   | accounts.bak     |
| transactions.dat | transactions.bak |

---

## ▶️ Execução

```bash
cobc -x BANK-SYSTEM-D24.cob
./BANK-SYSTEM-D24
