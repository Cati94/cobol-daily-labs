🎯 Objectives

Learn how to define and open a file

Read records from a sequential file

Display the contents of a file

Understand the basic COBOL file structure

🧠 Concept

Create a COBOL program that reads a text file containing customer names and displays each name on the screen.

The program will:

Open a file in INPUT mode.

Read each record sequentially.

Display the record on screen.

Stop when reaching the end of file.

This reflects how many legacy systems process batch data such as customer lists, payroll records, or transaction logs.
